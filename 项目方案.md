# 同益教育网站第一版方案

## 1. 项目目标

### 核心目标
- 建立“同益未来教育学社”的品牌可信度
- 清晰展示“人生规划与复盘学习共修营”课程信息
- 让私域用户在手机端快速完成了解并添加微信咨询

### 目标用户
- 来自私域流量池的潜在学员
- 对职业发展、人生规划、自我复盘有兴趣的人群
- 在重大转型、升学、就业、创业节点上存在决策焦虑的人群

### MVP 范围
- 一个移动端优先的品牌展示首页
- 完整承载品牌介绍、课程价值、导师信息、课程安排、报名方式
- 页面底部与悬浮 CTA 都指向微信咨询
- 保留后续扩展为多页面内容站的结构

## 2. 信息架构

### 当前第一版
- 首页
  - 顶部品牌识别
  - 首屏价值主张
  - 用户痛点
  - 课程价值与特色
  - 适合人群与学习收获
  - 导师阵容
  - 可信背书
  - 课程安排与权益
  - 报名流程
  - FAQ
  - 决策收口
  - 关于我们
  - 微信咨询 CTA

### 第二阶段可扩展页面
- 课程详情页
- 导师详情页
- 往期回顾 / 学员反馈页
- 品牌文章 / 活动动态页

## 3. 首页线框级方案

### Section 1: Hero 首屏
- Logo
- 品牌名 + 英文副标题
- 一句核心价值主张
- 简短课程说明
- 两个 CTA：了解课程 / 添加微信

### Section 2: 用户痛点
- 3 到 4 个典型问题
- 强调“从解决问题转向设计人生”

### Section 3: 课程价值
- 三位导师的组合优势
- 小班共修与成长董事会机制
- 审核制与限额带来的稀缺感和质量感

### Section 4: 导师介绍
- 3 位导师卡片
- 每位导师聚焦一个核心方向

### Section 5: 课程信息
- 上课形式
- 时间安排
- 价格与退款承诺
- 学员权益

### Section 6: 关于我们
- 同益未来教育学社简介
- 品牌气质与教育理念

### Section 7: 转化区
- 微信二维码
- 联系人、电话、微信号
- 强化立即咨询动作

## 4. 视觉方向

- 主色：暖金黄 + 米白色
- 风格关键词：高级、理性、克制留白、可信
- 页面氛围：杂志化排版 + 柔和渐变背景 + 轻量滚动动效
- 设备优先级：手机优先，桌面端自适应增强

## 5. 技术方案

- 技术形态：静态站点
- 首版建议：`HTML + CSS + JS`，零框架、低部署成本、便于快速上线
- 部署建议：GitHub Pages / Netlify / Vercel 均可
- 后续若持续更新频繁，可迁移到 Astro 或 Next.js

## 6. 分阶段执行计划

### 阶段 1：先上线
- 完成首页视觉与内容整合
- 接入 logo 与微信二维码
- 完成手机端适配与基础 SEO
- 补齐 `README`、`robots.txt`、`sitemap.xml` 等部署基础文件
- 补齐 GitHub Pages 自动部署工作流
- 输出可直接部署的静态文件

### 阶段 2：补强信任感
- 增加导师详情、往期反馈、活动照片
- 增加 FAQ、适合人群、报名流程等决策辅助信息
- 优化品牌叙事和转化文案

### 阶段 3：持续更新
- 扩展内容页结构
- 建立可维护的数据化内容录入方式
- 补充统计分析与更细致的转化追踪
